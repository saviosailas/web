Self closing tags / Void tags

1. The Thematic Break (Horizontal Rule)

<hr> - HTML5 format.

<hr/> - XHTML format (It is no longer needed, as most browsers support HTML5.)

2. Line Break

<br> - HTML5 format.

<br/> - XHTML format (It is no longer needed, as most browsers support HTML5.)

Always use <p> tag for adding one blank line after current text.
<br> are useful in situations like breaking lines of poem.
